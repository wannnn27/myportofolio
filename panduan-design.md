# Software Engineering Portfolio Design Guide

## Overview

Panduan ini dibuat untuk membantu menciptakan portfolio software engineering yang terlihat profesional, modern, recruiter-friendly, dan tidak tampak seperti desain AI-generated atau template generik.

Fokus utama portfolio:

* Software Engineering
* Full Stack Development
* Cloud Computing
* Artificial Intelligence
* Backend Engineering
* Mobile Development

---

# Design Principles

## 1. Prioritize Professionalism

Portfolio harus terasa seperti milik software engineer profesional, bukan landing page startup yang terlalu ramai.

Gunakan:

* Layout yang clean
* Typography yang jelas
* Struktur informasi yang rapi
* Visual hierarchy yang kuat
* White space yang cukup

Hindari:

* Efek berlebihan
* Dekorasi random
* Terlalu banyak gradient
* Tampilan cyberpunk berlebihan
* Animasi yang mengganggu

---

# Color Guidelines

## Recommended Colors

Gunakan warna clean dan profesional.

### Primary Colors

* Black
* White
* Charcoal
* Deep Navy
* Slate Gray

### Accent Colors

Gunakan secara subtle:

* Soft Blue
* Muted Cyan
* Steel Blue

## Avoid These Colors

Jangan gunakan:

* Pink
* Peach
* Pastel colors
* Rainbow gradients
* Neon colors berlebihan
* Saturated purple
* Feminine palettes

## Color Usage Rules

* Maksimal 1–2 accent colors
* Background tetap clean
* Gunakan kontras yang nyaman dibaca
* Jangan gunakan glow berlebihan
* Hindari gradient yang terlalu mencolok

---

# Typography Guidelines

## Font Style

Gunakan modern sans-serif fonts.

Recommended:

* Inter
* SF Pro Display
* Geist
* Poppins
* Manrope

## Typography Hierarchy

### Heading

* Bold
* Clear
* Large
* Strong visual hierarchy

### Body Text

* Readable
* Medium spacing
* Jangan terlalu kecil

### Important Notes

* Hindari font futuristik
* Hindari font decorative
* Gunakan maksimal 2 font families

---

# Layout Guidelines

## Layout Style

Gunakan:

* Grid-based layout
* Balanced whitespace
* Consistent spacing
* Clean alignment
* Structured sections

## Recommended Spacing

* Large padding between sections
* Comfortable margins
* Consistent card spacing

## UI Components

Gunakan:

* Clean cards
* Subtle shadows
* Thin borders
* Minimal hover effects
* Rounded corners secukupnya

Hindari:

* Glassmorphism berlebihan
* Heavy blur effects
* Neon borders
* Floating random elements

# Portfolio Structure

## 1. Hero Section

### Purpose

Membangun first impression profesional.

### Include

* Nama lengkap
* Role/title
* Short professional tagline
* Professional photo placeholder
* CTA buttons

### Example Roles

* Software Engineer
* Full Stack Developer
* Cloud & AI Enthusiast
* Backend Developer

### Hero Design Tips

* Clean background
* Minimal illustration
* Fokus pada readability
* Jangan terlalu banyak dekorasi

---

# 2. About Me Section

# Avoid AI-Generated Look

## Common AI-Generated Problems

Hindari:

* Terlalu banyak gradient
* Layout terlalu sempurna namun tidak realistis
* Neon glow berlebihan
* Random futuristic elements
* Floating shapes tidak berguna
* Typography tidak konsisten
* UI terlalu glossy
* Overdesigned components

## Make It Feel Real

Gunakan:

* Realistic spacing
* Practical layouts
* Professional typography
* Consistent visual rhythm
* Functional UI
* Subtle effects only

Portfolio harus terasa seperti:

* Dibuat designer profesional
* Digunakan engineer sungguhan
* Siap dipresentasikan ke recruiter

* [ ] Layout clean dan modern
* [ ] Tidak terlihat AI-generated
* [ ] Color palette profesional
* [ ] Typography konsisten
* [ ] White space cukup
* [ ] UI tidak terlalu ramai

## Recruiter Experience

* [ ] Mudah discan dalam 30 detik
* [ ] Projects mudah dipahami
* [ ] Skill hierarchy jelas
* [ ] Achievement terlihat menonjol
* [ ] Professional branding kuat

## Technical Quality

* [ ] Konsisten spacing
* [ ] Konsisten card design
* [ ] Responsive layout
* [ ] High readability
* [ ] Modern but realistic
